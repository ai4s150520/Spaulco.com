# Generate Favicons from Logo.png

## Quick Steps:

1. Go to: https://realfavicongenerator.net

2. Upload: C:\Users\USER\Desktop\Subrato_Paul_Website\ca-firm-kolkata\public\favicon.png

3. Download the generated package

4. Extract these files to the `public/` folder:
   - favicon.ico
   - favicon-16x16.png
   - favicon-32x32.png
   - apple-touch-icon.png
   - android-chrome-192x192.png
   - android-chrome-512x512.png

5. Run: npm run build

6. Deploy your website

Your logo will appear in Google search results after Google re-crawls your site (24-48 hours).
